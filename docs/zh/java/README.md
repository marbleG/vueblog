# Java

这里是我的 Java 学习笔记，整理了 Java 核心知识点、JVM、并发编程、Spring 生态等内容，适合后端开发复习查阅。

## 分类导航

### Java 新特性
- Java 8：Lambda、Stream、Optional、新日期时间 API
- Java 9 - 11：模块系统、局部变量类型推断、ZGC、HttpClient
- Java 17：密封类、模式匹配、记录类、文本块，当前 LTS 版本

### JVM
- 运行时数据区域：程序计数器、虚拟机栈、堆、方法区
- 垃圾回收：算法、垃圾回收器、调优参数
- 类加载机制：双亲委派、打破双亲委派

### 并发编程
- 基础概念、线程池、锁机制、同步工具类、原子类

### Spring 生态
- IOC 容器、AOP、Spring Boot 自动配置、常用注解总结

### ORM & 数据库
- JDBC 基础、MyBatis 原理、JPA/Hibernate

### 网络编程
- IO/BIO/NIO 对比、Netty 核心原理

### 集合框架
- 概述、List/Set 对比、HashMap 原理

所有内容整理基于 OpenJDK 官方文档和主流技术资料，保证准确。
